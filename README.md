# dotfiles
A repo for all my config files
## Setup
```bash
git clone https://github.com/abpwrs/dotfiles.git ~/.files/ 
cd ~/.files && ./setup
source ~/.bashrc # if using bash
source ~/.zshrc # if using zsh
vim ~/.vimrc
```

## Configs
* Bash                   
* Zsh                    
* Git                    
* vim (https://github.com/aaronbieber/dotvim)                    
* bin (my scripts)
